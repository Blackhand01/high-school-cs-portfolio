# High School CS Portfolio

A comprehensive collection of high school computer science projects organized by topic and skill level. This portfolio showcases coursework in C++ programming and Arduino embedded systems development.

## üìö Repository Structure

### 1. **C++ Fundamentals** (`01-cpp-fundamentals/`)
Introduction to C++ programming with essential algorithms and basic data types.
- **Basic Algorithms**: Armstrong numbers, Fibonacci series, mathematical operations
- **Geometry**: Triangle/rectangle/circle calculations, coordinate plane operations
- **Conversions**: Time and unit conversions

### 2. **C++ Intermediate** (`02-cpp-intermediate/`)
Core data structures and intermediate programming concepts.
- **Arrays & Vectors**: Vector operations, sorting, sequence analysis
- **Quadratic Equations**: Mathematical equation solvers
- **Statistics**: Data analysis and calculations

### 3. **C++ Strings & Data Structures** (`03-cpp-strings-data/`)
Advanced string manipulation and structured data handling.
- **Strings**: Text processing and manipulation
- **Fiscal Code Validator**: Italian fiscal code generation and validation (multiple iterations showing progression)
- **Structures**: Complex data organization and student management systems

### 4. **C++ Matrices** (`04-cpp-matrices/`)
Matrix operations and two-dimensional data structures.
- **Magic Squares**: Algorithm implementation and variations
- **Matrix Operations**: General matrix computations

### 5. **C++ Games** (`05-cpp-games/`)
Game development projects demonstrating algorithmic thinking.
- **Tombola Lottery**: Italian lottery simulator with multi-player support
- **Frame Game**: Border-based game with various player and screen configurations

### 6. **Arduino Projects** (`06-arduino-projects/`)
Embedded systems development with Arduino microcontrollers.
- **LED Control**: Digital and PWM-based LED operations
- **Sensors**: Analog sensor reading (photoresistors, potentiometers, buzzers)
- **Infrared**: TV-B-Gone remote control and IR code management
- **Security**: Rubber ducky security tools

### 7. **Documentation & Resources** (`07-notes-docs/`)
Learning materials, references, and external documentation.
- **C++ Learning Notes**: Personal notes and reference materials
- **External Documents**: PDFs and reports on Linux, networking, and projects
- **Tools Configuration**: AStyle formatting and development tools

### Maintenance
Archive folder for obsolete and experimental code from previous iterations.

## üìñ How to Use

### Running C++ Programs
```bash
# Using Dev-C++ or g++
g++ filename.cpp -o program_name
./program_name

# Or compile with Dev-C++ IDE
```

### Running Arduino Projects
1. Open Arduino IDE
2. Select appropriate board and COM port
3. Load `.ino` file from `06-arduino-projects/`
4. Upload to board
